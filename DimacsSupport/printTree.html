Preorder Tree Traversal - Line by Line Explanation

void printPreorder(Node *root)
Function Declaration: This declares a function named printPreorder that takes a pointer to a Node structure as its parameter. The function returns void (no return value). The parameter root represents the current node being visited in the tree.
{
Opening Brace: Marks the beginning of the function body where all the logic is contained.
if (root == NULL)
Base Case Check: This is the stopping condition for the recursion. It checks whether the current node pointer is NULL (i.e., the node doesn't exist). This happens when:
  • We've reached a leaf node and try to visit its non-existent children
  • The tree itself is empty (root is NULL from the start)
Why this is critical: Without this check, the program would crash with a segmentation fault when trying to access NULL->value.
return;
Early Return: If the node is NULL, immediately exit this function call without doing anything. This terminates the current recursive branch and returns control to the calling function.
printf("%c ", root->value);
Process Current Node: Print the value stored in the current node followed by a space. The %c format specifier indicates the value is a character. This line executes before visiting children, which is what makes this a preorder traversal.
Preorder Definition: Visit order is Root → Left → Right. The root is processed first.
printPreorder(root->left);
Recursive Call - Left Subtree: Make a recursive call to process the entire left subtree. This call:
  • Accesses the left child of the current node using root->left
  • Treats that left child as the new root of its own subtree
  • Recursively prints all nodes in the left subtree in preorder fashion
  • Only returns when the entire left subtree has been traversed
printPreorder(root->right);
Recursive Call - Right Subtree: After the left subtree is completely processed, make a recursive call to process the entire right subtree. This call:
  • Accesses the right child of the current node using root->right
  • Treats that right child as the new root of its own subtree
  • Recursively prints all nodes in the right subtree in preorder fashion
}
Closing Brace: Marks the end of the function. When execution reaches here, the function returns control to its caller (either the previous recursive call or the original caller).
Example Execution:
For a tree:    A
            / \
           B   C

Output: A B C

Execution Flow:
1. Print A (root)
2. Recursively visit left subtree → Print B
3. Recursively visit right subtree → Print C
Time Complexity: O(n) where n is the number of nodes, since each node is visited exactly once.
Space Complexity: O(h) where h is the height of the tree, due to the recursive call stack.
