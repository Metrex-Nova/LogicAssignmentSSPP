Explanation of freeTree Function

void freeTree(Node *root)
Declares a function named freeTree that takes a pointer to the root node of a tree. 
{
Begins the function body. 
    if (root == NULL)
Checks if the current node pointer is NULL (i.e., the tree or subtree is empty). 
        return;
If the node is NULL, exit the function without doing anything. 
    freeTree(root->left);
Recursively calls freeTree on the left child of the current node. 
    freeTree(root->right);
Recursively calls freeTree on the right child of the current node. 
    free(root);
Frees the memory allocated for the current node after its children have been freed. 
}
