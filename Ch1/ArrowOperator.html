Chapter: The -> Operator in C

The -> operator is used in C for accessing structure members through a pointer to a struct. It provides a convenient way to refer to the fields of a structure without explicit dereferencing.

1. What does -> do?

If you have a pointer to a struct (for example, ptr) and you want to access a member (such as age), you use: 

ptr->age

This is equivalent to: 

(*ptr).age

Parentheses are required because the dot operator (.) has higher precedence than the asterisk (*).

2. Why use ->?

The -> operator is more concise and readable than (*ptr).member.

3. Example

struct Person {
        int age;
        char name[20];
    };

    struct Person p1 = {30, "Alice"};
    struct Person *ptr = &p1;

    // Accessing age using pointer
    ptr->age = 21;  // Sets the 'age' field of p1 to 21

    printf("Age: %d\n", ptr->age);  // Prints 21

4. Summary Table

Expression Meaning
ptr->age Access age member via pointer
(*ptr).age Dereference pointer, then access member

Use -> whenever you have a pointer to a struct and want to access its members.
