Structures in c

This lesson covers essential C programming concepts relevant to our codebase: struct, structure, pointer, alias (typedef), and variable types, with clear examples. At the end, it breaks down our actual code structures, and lists related concepts that we should know.

1. struct in C

A struct lets you group related variables (possibly of different types) under one name, as a new composite type. Think of it like a record or a simple object. 

// Define a struct for a student
struct Student {
    char name;
    int age;
    float grade;
};

Declaring and using: 

struct Student s1;
s1.age = 20;
s1.grade = 8.9;

2. Structure vs struct

The term structure is the abstract concept, and struct is the concrete C keyword for defining one. Whenever you use a type defined by struct, you are creating a structure. 

// Struct type definition (template)
struct Point { int x, y; };

// Creating an instance (structure)
struct Point p1 = {0, 1};

typedef struct {
    char variable;        // e.g., 'P'
    int value;            // 0 (false) or 1 (true)
} TruthAssignment;

// DIMACS clause: list of literals (variables or negated variables)
typedef struct {
    int* literals;        // e.g., {1, -2, 3}
    int size;             // number of literals in clause
} Clause;

// DIMACS CNF formula: set of clauses
typedef struct {
    Clause* clauses;      // Array of Clause structs
    int numClauses;       // How many clauses
    int numVars;          // How many variables
} DIMACSFormula;

// Mapping from char variable to DIMACS int variable
typedef struct {
    char charVar;         // e.g., 'A'
    int intVar;           // e.g., 1
} VarMapping;
  • Node: Used for building a binary parse tree: every logic operator or variable is a node; left/right point to sub-nodes.
  • TruthAssignment: Associates a variable name (e.g., 'P') with a truth value (0 for false, 1 for true).
  • Clause: Represents a single clause for SAT/CNF, i.e., a disjunction of literals (variables that can be positive or negative). Stored as an array of int where sign marks negation.
  • DIMACSFormula: Represents the whole logic formula in CNF: an array of Clauses, and metadata about clause and variable counts.
  • VarMapping: Association table for mapping between character variable names (charVar, like 'P') and integer equivalents required by SAT solvers (intVar).
