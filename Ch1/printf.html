Chapter: printf Statement in C

printf() is one of the most important functions in C for displaying output. It allows you to print text, numbers, variables, and even memory addresses to the screen, using a format string to control how the output appears.

1. Basic Usage

printf("Hello, World!\n");

This prints the text Hello, World! followed by a newline.

2. Format Specifiers

Format specifiers are placeholders in the format string that tell printf how to display each value. Each specifier starts with % and is followed by a character indicating the type.

SpecifierTypeExample
%dint (decimal)printf("%d", 42); → 42
%ffloat/doubleprintf("%f", 3.14); → 3.140000
%ccharprintf("%c", 'A'); → A
%sstringprintf("%s", "Hello"); → Hello
%ppointer (address)printf("%p", ptr); → 0x7ffee1a8
%uunsigned intprintf("%u", 3000000000U); → 3000000000
%xhexadecimalprintf("%x", 255); → ff
%ooctalprintf("%o", 8); → 10

3. Printing Multiple Values

int a = 5;
float b = 2.5;
char c = 'Z';
printf("a = %d, b = %.2f, c = %c\n", a, b, c);

Output: a = 5, b = 2.50, c = Z

Note: %.2f prints 2 decimal places.

4. Escape Sequences

Special characters can be printed using escape sequences:

  • \n : Newline
  • \t : Tab
  • \\ : Backslash
  • \" : Double quote
printf("Line1\nLine2\tTabbed\n");

5. Printing Pointers and Addresses

int x = 10;
int *p = &x;
printf("Address of x: %p\n", (void*)p);
printf("Value at p: %d\n", *p);

Output: Shows the memory address and the value stored at that address.

6. Formatting Output

You can control width, precision, and alignment:

printf("|%10d|\n", 123);   // Right-align in 10 spaces
printf("|%-10d|\n", 123);  // Left-align in 10 spaces
printf("%.3f\n", 3.14159); // 3 decimal places

7. Common Mistakes

  • Mismatch between format specifier and variable type (e.g., using %d for a float).
  • Forgetting to include \n for newlines.
  • Not casting pointers to void* when printing addresses.

8. Complete Example

#include 

int main() {
    int i = 42;
    float f = 3.1415;
    char ch = 'A';
    char str[] = "Hello";
    int *ptr = &i;

    printf("Integer: %d\n", i);
    printf("Float: %.2f\n", f);
    printf("Char: %c\n", ch);
    printf("String: %s\n", str);
    printf("Pointer: %p\n", (void*)ptr);
    printf("Hex: %x\n", i);
    printf("Octal: %o\n", i);
    return 0;
}

10. Key Takeaways

  • printf() is essential for debugging and displaying results.
  • Always match format specifiers to variable types.
  • Use escape sequences for special formatting.
  • Control output width and precision for neat display.
