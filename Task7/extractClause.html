Explanation: extractClauses Function

void extractClauses(Node *root, Node **clauses, int *count, int maxClauses)
{
    if (root == NULL || *count >= maxClauses)
        return;

    if (root->value == '*')
    {
        extractClauses(root->left, clauses, count, maxClauses);
        extractClauses(root->right, clauses, count, maxClauses);
    }
    else
    {
        clauses[(*count)++] = root;
    }
}

void extractClauses(Node *root, Node **clauses, int *count, int maxClauses)
Defines a function named extractClauses that takes a pointer to the root of a CNF subtree, an array of clause pointers, a pointer to the number of clauses collected, and the maximum number of clauses allowed. 
if (root == NULL || *count >= maxClauses)
    return;
Checks if the root is NULL (no node to process) or if the maximum number of allowed clauses has been reached. If either is true, the function returns immediately without further processing. 
if (root->value == '*')
Checks if the current node represents a logical AND operator (*), which indicates multiple clauses in a CNF expression. 
{
    extractClauses(root->left, clauses, count, maxClauses);
    extractClauses(root->right, clauses, count, maxClauses);
}
If the node is an AND node, recursively extract clauses from the left and right subtrees. This effectively splits a CNF expression of the form (A * B) into two clauses A and B. 
else
{
    clauses[(*count)++] = root;
}
If the node is not an AND operator, it represents a single clause. The function adds the node (clause) to the clauses array and increments the clause count.
