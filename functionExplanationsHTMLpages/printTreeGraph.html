// Print tree level by level (root at top, indented for binary tree structure)
void printTreeRooted(Node *root, int level)

**Function Declaration:** This declares a function named printTreeRooted that takes a pointer to a Node structure named root and an integer level as parameters. The function returns void (no return value). The root parameter represents the current node being visited in the binary tree, and level indicates the depth or level of the node for indentation purposes.

{

**Opening Brace:** Marks the beginning of the function body where all the logic is contained.

if (root == NULL)

**Base Case Check:** This is the stopping condition for the recursion. It checks whether the current node pointer root is NULL, indicating that the node doesn’t exist. This happens when a leaf node’s children or an empty subtree is encountered.

return;

**Base Case Return:** If root is NULL, the function terminates immediately, preventing further recursion and avoiding access to non-existent nodes.

for (int i = 0; i < level; i++)

**Indentation Loop Initialization:** This starts a for loop with an integer variable i initialized to 0. The loop continues as long as i is less than level, incrementing i by 1 each iteration. This loop is used to print the appropriate number of spaces for indentation based on the node’s level in the tree.

printf("    ");

**Print Indentation:** Inside the loop, this prints four spaces (" ") for each iteration, creating an indentation that visually represents the tree’s hierarchical structure, with the root at the top and deeper levels indented further.

printf("%c\n", root->value);

**Print Node Value:** This prints the value of the current node (root->value) as a character (using %c) followed by a newline (\n). This displays the node’s value at the correct indentation level, forming a level-by-level representation of the binary tree.

printTreeRooted(root->left, level + 1);

**Recursive Left Subtree Call:** This recursively calls printTreeRooted on the left child of the current node (root->left), increasing the level by 1 to indent the left subtree one level deeper.

printTreeRooted(root->right, level + 1);

**Recursive Right Subtree Call:** This recursively calls printTreeRooted on the right child of the current node (root->right), also increasing the level by 1 to indent the right subtree one level deeper.

}

**Closing Brace:** Marks the end of the printTreeRooted function body.

{

**Base Case Return:** If root is NULL, the function terminates immediately, preventing further processing of non-existent nodes.

**Left Child Existence Check:** This checks if the left child of the current node (root->left) exists (i.e., is not NULL). The implicit conversion of a pointer to a boolean checks for non-nullness.

**Print Left Edge:** If a left child exists, this prints the current node’s value (root->value) and the left child’s value (root->left->value) connected by " --> ", followed by a newline (\n). This represents a directed edge from the parent to the left child in the graph.
