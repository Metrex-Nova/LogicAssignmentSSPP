Explanation: createNode Function

Node* createNode(char value) {
    Node* newNode = (Node*)malloc(sizeof(Node));
    newNode->value = value;
    newNode->left = NULL;
    newNode->right = NULL;
    return newNode;
}
  • Node* createNode(char value): Declares a function named createNode that takes a character (value) and returns a pointer to a Node.
  • Node* newNode = (Node*)malloc(sizeof(Node));: Dynamically allocates memory for a new Node struct using malloc. The result of malloc is cast to the type Node*. This allocation is done on the heap, so the node can persist after the function returns.
  • newNode->value = value;: Assigns the input character value to the value field of the new node.
  • newNode->left = NULL;: Sets the left pointer of the node to NULL, indicating it does not yet have a left child.
  • newNode->right = NULL;: Sets the right pointer of the node to NULL, meaning it does not yet have a right child.
  • return newNode;: Returns the pointer to the newly created node, allowing other parts of the program to use it.

Memory Allocation Notes

  • malloc(sizeof(Node)) allocates enough space in memory for one Node. This is done at runtime, not compile time, providing flexibility for dynamic data structures like trees and linked lists.
  • Always check if malloc returns NULL before using the pointer, to avoid crashes if memory allocation fails: 
    if (newNode == NULL) {
    // handle error
}
  • Remember to free() each allocated Node when it's no longer needed, preventing memory leaks.
