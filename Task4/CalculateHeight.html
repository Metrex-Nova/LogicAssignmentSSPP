Explanation: Calculate Height Function

int calculateHeight(Node* root) {
    if (root == NULL) {
        return -1;
    }
    
    int leftHeight = calculateHeight(root->left);
    int rightHeight = calculateHeight(root->right);
    
    return 1 + (leftHeight > rightHeight ? leftHeight : rightHeight);
}
Computes the height of a binary parse tree rooted at root. The height is the length of the longest path from the root node down to a leaf node.

Line-by-Line Explanation

  • if (root == NULL) { return -1; }
    Base case: an empty tree (null pointer) has height -1. This convention means a leaf node with no children has height 0.
  • int leftHeight = calculateHeight(root->left);
    Recursively compute the height of the left subtree.
  • int rightHeight = calculateHeight(root->right);
    Recursively compute the height of the right subtree.
  • return 1 + (leftHeight > rightHeight ? leftHeight : rightHeight);
    Return 1 plus the maximum of left and right subtree heights: - 1 accounts for the current node's level. - The max chooses the taller subtree to determine total height.

Key Concepts

  • Recursion: Each call processes subtrees independently and combines their results.
  • Height Definition: Number of edges on the longest downward path from root to leaf. Returning -1 for NULL nodes aligns leaf nodes with height 0.
  • Binary Tree: Function works for any binary tree, including parse trees representing logical expressions.

Example Usage

int height = calculateHeight(root);
printf("Height of tree: %d\n", height);

Outputs the height of the tree rooted at root.
